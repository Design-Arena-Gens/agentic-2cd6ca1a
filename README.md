# Instagram Analyzer

A Next.js application for analyzing Instagram account metrics and comparing multiple accounts side by side.

## Features

- Analyze multiple Instagram accounts simultaneously
- Display key metrics: followers, average views, engagement rate, category, and location
- Sortable data table for easy comparison
- Export results to CSV or Excel format
- Responsive, modern UI with Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
